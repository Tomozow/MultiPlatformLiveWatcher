YouTubeの連携には、APIキーとクライアントIDが必要です：

APIキーとクライアントIDの取得方法:

  1. Google Cloud Consoleにアクセスし、Googleアカウントでログイン
  2. ページ上部のプロジェクト選択メニューから「新しいプロジェクト」を選択して作成
  3. 左側のナビゲーションメニューから「APIとサービス」→「ライブラリ」を選択
  4. 検索ボックスで「YouTube Data API v3」を検索し、選択して「有効にする」をクリック
  5. 「APIとサービス」→「認証情報」を選択
  6. 「認証情報を作成」→「APIキー」をクリックしてAPIキーを作成
  7. 作成されたAPIキーをコピーして上記の「API Key」欄に入力
  8. 次に「認証情報を作成」→「OAuth クライアント ID」を選択
  9. アプリケーションの種類として「ウェブアプリケーション」を選択
  10. 名前を入力（例: 「配信通知拡張機能」）
  11. 「承認済みのリダイレクトURI」セクションで「URIを追加」をクリック
  12. 上部に表示されている「リダイレクトURI」を正確にコピー＆ペースト
  13. 「作成」ボタンをクリックしてOAuthクライアントIDを作成
  14. 作成されたクライアントIDをコピーして上記の「クライアントID」欄に入力
  15. 重要: セキュリティのため、APIキーに以下の制限を設定してください：
    • 作成したAPIキーの横にある鉛筆アイコン（編集）をクリック
    • 「アプリケーションの制限」セクションで「HTTPリファラー」を選択
    • 「ウェブサイトの制限」項目で「項目を追加」をクリック
    • 「chrome-extension://<あなたの拡張機能ID>/*」と入力（拡張機能IDは、Chromeの拡張機能管理ページで確認できます）
    • 「APIの制限」セクションで「APIを制限」を選択
    • ドロップダウンリストから「YouTube Data API v3」にチェックを入れる
    • 「保存」ボタンをクリックして設定を完了

拡張機能の認証設定:

Googleアカウントでの認証を行うには、APIキーとクライアントIDを入力後、「認証する」ボタンをクリックしてください。 ポップアップウィンドウが表示され、YouTubeアカウントへのアクセス許可を求められます。

API使用制限についての注意事項:

  • YouTube Data API v3には無料枠で1日あたり10,000ユニットの制限があります
  • 配信データの取得には約100ユニット/リクエストが使用されます
  • 更新頻度が高いと、APIクォータをすぐに使い切る可能性があるため注意してください
  • APIクォータの確認は「APIとサービス」→「ダッシュボード」→「クォータ」から確認できます
  • クォータを超えた場合は、Google Cloud Consoleで課金を有効にすることで追加のクォータを購入できます

  3. 通知したい配信者のユーザーIDを追加（例: username）
